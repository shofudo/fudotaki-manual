1. せきに あんない（いりぐち から せきまで）1. 席に案内（入口から席まで）/1. सिटमा मार्गदर्शन (प्रवेशद्वारदेखि सिटसम्म)

しょくじばしょ の いりぐちで おきゃくさま を むかえ、あいさつ を します。食事場所の入口でお客様を迎え、挨拶をします。/भोजनस्थलको प्रवेशद्वारमा पाहुनालाई स्वागत गरेर अभिवादन गर्नुहोस्。

「いらっしゃいませ。」「いらっしゃいませ。」/"स्वागतम्।"

へや の なまえ を かくにんし、ようい した せき に ごあんない します。部屋の名前を確認し、用意した席にご案内します。/कोठाको नाम पुष्टि गरेर, तयार गरिएको सिटमा लैजानुहोस्。

「おせき に ごあんない いたします。」「お席にご案内いたします。」/"म तपाईंलाई सिटसम्म मार्गदर्शन गर्छु।"

※ のみもの を うかがう スタッフ が いない ばあい※ 飲み物をうかがうスタッフがいない場合/※ पेय सोध्ने कर्मचारी तत्काल उपलब्ध नभएको अवस्थामा

「ただいま ごあんない させて いただきますので、しょうしょう おまちください。」「ただいまご案内させていただきますので、少々お待ちください。」/"अहिले मार्गदर्शन गर्छौं, कृपया केही बेर पर्खनुहोस्।"

2. せきに ついた あと の ごあんない2. 席に着いたあとのご案内/2. सिटमा बसेपछि दिने जानकारी

「ごせつめい を させて いただきます。」「ご説明をさせていただきます。」/"म तपाईंलाई जानकारी दिन्छु।"

「こちらは しょくぜんしゅ の ゆずしゅ で ございます。」「こちらは食前酒の柚子酒でございます。」/"यो भोजनअघि पिउने युजु मदिरा हो।"

※ ノンアルコール の かた には 「ざくろす」 を ごようい しております。※ ノンアルコールの方には「ざくろ酢」をご用意しております。/※ मदिरा नपिउने पाहुनाका लागि "अनार सिरका पेय" उपलब्ध छ。

「こちらに ほんじつ の こんだて と、のみもの の メニュー を ごようい しております。」「こちらに本日の献立と、飲み物のメニューをご用意しております。」/"यहाँ आजको भोजन सूची र पेय मेनु तयार छ।"

「あたたかい おちゃ と おみず は、むりょう で ごようい できます。おきまり に なりましたら、おしらせ ください。」「温かいお茶とお水は、無料でご用意できます。お決まりになりましたら、お知らせください。」/"तातो चिया र पानी निःशुल्क उपलब्ध छ。 निर्णय भएपछि कृपया बताउनुहोस्。"

※ タイミング を みて、のみもの の ちゅうもん を うかがい オーダー します。むりょう の おちゃ は きほん 「ほうじちゃ」。おみず は こおり の あり／なし を かくにん。※ タイミングを見て飲み物の注文をうかがい、オーダーします。無料のお茶は基本「ほうじ茶」。お水は氷のあり・なしを確認します。/※ उपयुक्त समयमा पेयको अर्डर लिनुहोस्。 निःशुल्क चिया सामान्यतया होउजिचा हो。 पानीमा बरफ चाहिन्छ कि चाहिँदैन, सोधेर पुष्टि गर्नुहोस्。

3. ステーキ タレ の せつめい（ステーキプラン／わぎゅうかいせき）3. ステーキたれの説明（ステーキプラン／和牛会席）/3. स्टेक ससको व्याख्या (स्टेक प्लान / वाग्यु काइसेकि)

  1. ステーキ の まえ の りょうり（あげもの）を ていきょう するとき に、ステーキ の タレ を いっしょ に もって いく。ステーキの前の料理（揚げ物）を提供するときに、ステーキのたれを一緒に持っていきます。/स्टेकअघिको परिकार (तारेको परिकार) लैजाँदा, स्टेकको सस पनि सँगै लिएर जानुहोस्。
  2. りょうり を ていきょう して せつめい した あと に、ステーキ の タレ を ていきょう して せつめい を する。料理を提供して説明したあとに、ステーキのたれを提供して説明します。/परिकार राखेर व्याख्या गरेपछि, स्टेकको सस पनि राखेर त्यसको व्याख्या गर्नुहोस्。
  3. 「こちらは ステーキ の タレ に なります。」「こちらはステーキのたれになります。」/"यो स्टेकको सस हो।"
  4. 「いちばん うえ は 『ショウガス』。ショウユ は かさいもり でも ごていきょう した 『もろみしょうゆ』。『モジオ』 と 『クロコショウ』 も ごようい しておりますので、この あと の ステーキ に ごりよう ください。」「一番上は『ショウガス』。醤油は火菜盛りでもご提供した『もろみ醤油』。『藻塩』と『黒胡椒』もご用意しておりますので、このあとのステーキにご利用ください。」/"सबैभन्दा माथि अदुवाको सस छ。 सोया सस भने पहिल्यै परोसेको मोरोमी शौयु हो। मोशिओ र कालो मरिच पनि तयार छन्, कृपया त्यसपछिको स्टेकसँग प्रयोग गर्नुहोस्।"

4. ごはん の うかがいかた（ニモノ／ステーキ の あと）4. ご飯のうかがい方（煮物／ステーキのあと）/4. भातको समय सोध्ने तरिका (निमोनो / स्टेकपछि)

ニモノ の せつめい の あと、つぎ の ように うかがいます。煮物の説明のあと、次のようにうかがいます。/निमोनोको व्याख्यापछि, यसरी सोध्नुहोस्。

「このあと、ごはん と なりますが、ごようい させて いただいても よろしい でしょうか？」「このあと、ご飯となりますが、ご用意させていただいてもよろしいでしょうか？」/"अब भात ल्याउने समय हुन्छ, के अहिले तयार गरेर ल्याउन मिल्छ?"

※ りょうり や おさけ が のこって いる とき は、ききかた を かえる※ 料理やお酒が残っているときは、聞き方を変えます。/※ खाना वा मदिरा अझै बाँकी छ भने, सोध्ने तरिका बदल्नुहोस्。

「このあと、ごはん と なりますが、すこし じかん を あけた ほうが よろしい でしょうか？」「このあと、ご飯となりますが、少し時間を空けたほうがよろしいでしょうか？」/"अब भात ल्याउने बेला हुन्छ, के अलि समय पछि ल्याउनु राम्रो हुन्छ?"

（あいだ を あけて ほしい と いわれたら）（間を空けてほしいと言われたら）/(यदि केही समय पर्खन भन्नुभयो भने)

「かしこまりました。ひつよう な とき に おこえがけ ください。」「かしこまりました。必要なときにお声がけください。」/"हुन्छ। आवश्यक परेमा कृपया बोलाउनुहोस्।"

※ ボタン の ある へや は 「ボタン を おしてください」 と つたえる。※ ボタンのある部屋は「ボタンを押してください」と伝えます。/※ कल बटन भएको कोठामा "कृपया बटन थिच्नुहोस्" भनेर बताउनुहोस्。

5. かんみ ていきょう まえ の かくにん5. 甘味提供前の確認/5. मिठाई दिनुअघि पुष्टि

ほうじちゃ を おもち し、げぜん を しながら ようす を うかがいます。ほうじ茶をお持ちし、下膳をしながら様子をうかがいます。/होउजिचा लिएर जाँदा, भाँडा झिक्दै पाहुनाको अवस्था हेर्नुहोस्。

「このあと、デザート の ごようい と なりますが、おもち しても よろしい でしょうか？」「このあと、デザートのご用意となりますが、お持ちしてもよろしいでしょうか？」/"अब डेजर्ट तयार छ, के म लिएर आउन सक्छु?"

「また、おやしょく よう に、めしあがって いただいた ごはん を おにぎり に して おもち しております が、ごようい いたします か？」「また、お夜食用に、召し上がっていただいたご飯をおにぎりにしてお持ちしておりますが、ご用意いたしますか？」/"साथै रातिको हल्का खानाका लागि बाँकी भातबाट ओनिगिरि बनाएर ल्याइदिन सक्छौं, के तयार गरौं?"
